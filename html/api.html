API Documentation

Description: Endpoint to generate an API token.

Parameters: JSON object with clientId.

Returns: JSON response with generated token.

Authorization: None required.

Description: Endpoint to add a new plant.

Parameters: JSON object with (place_of_collection, collection_id, hashtags, common_name, scientific_name, family, genus, species)

Returns: JSON response confirming success or failure.

Authorization: Bearer token in the Authorization header.

Description: Endpoint to fetch details of a specific plant.

Parameters: Query parameter 'plantId'.

Returns: JSON response with plant details.

Authorization: Bearer token in the Authorization header.

Description: Endpoint to fetch a collection of plants.

Parameters: None.

Returns: JSON response with an array of plant objects.

Authorization: Bearer token in the Authorization header.

Description: Endpoint to fetch all your plants.

Parameters: None.

Returns: JSON response with an array of all plant objects.

Authorization: Bearer token in the Authorization header.

Description: Endpoint to fetch comments for a specific resource.

Parameters: Query parameter 'resourceId'.

Returns: JSON response with comments.

Authorization: Bearer token in the Authorization header.

Description: Endpoint to fetch collections of resources.

Parameters: None.

Returns: JSON response with collections.

Authorization: Bearer token in the Authorization header.

Description: Endpoint to fetch badges earned by a user.

Parameters: Query parameter 'userId'.

Returns: JSON response with badges.

Authorization: Bearer token in the Authorization header.

Description: Endpoint to fetch a user's friends.

Parameters: Query parameter 'userId'.

Returns: JSON response with friends list.

Authorization: Bearer token in the Authorization header.

Description: Endpoint to delete a specific plant.

Parameters: Query parameter 'plantId'.

Returns: JSON response confirming deletion.

Authorization: Bearer token in the Authorization header.
